# MovieList

Application to get best movies

## Installation

Make sure you are using node V10.16.0

```bash
git clone https://github.com/RomanN-dev/MovieList.git

cd MovieList

yarn install or npm install

cd ios/

pod install

cd ..
```

## Usage

```
yarn run android
```
